i have folder inside it "index.theme" file using fish shell write script to read "index.theme" and extract all lines that between the line that start "#placesStart" and that start "#placesEnd"

You can use the following Fish shell script to read the index.theme file and extract all lines between the lines that start with #placesStart and #placesEnd:

fish
Copy
#!/usr/bin/fish

# Path to the index.theme file
set file "index.theme"

# Initialize a flag to indicate when to start/stop capturing lines
set capture false

# Read the file line by line
for line in (cat $file)
    # Check if the line starts with #placesStart
    if string match -q "#placesStart*" -- $line
        set capture true
        continue
    end

    # Check if the line starts with #placesEnd
    if string match -q "#placesEnd*" -- $line
        set capture false
        break
    end

    # If the capture flag is true, print the line
    if $capture
        echo $line
    end
end

Explanation:

  1. set file "index.theme": Specifies the path to the index.theme file.

  2. set capture false: A flag to determine whether we are within the desired range of lines.

  3. for line in (cat $file): Reads the file line by line.

  4. string match -q "#placesStart*" -- $line: Checks if the line starts with #placesStart. If so, it sets the capture flag to true.

  5. string match -q "#placesEnd*" -- $line: Checks if the line starts with #placesEnd. If so, it sets the capture flag to false and breaks out of the loop.

  6. if $capture: If the capture flag is true, it prints the line.

Usage:

  1. Save the script to a file, e.g., extract_places.fish.

  2. Make it executable: chmod +x extract_places.fish.

  3. Run it: ./extract_places.fish.

    # Check if the line starts with #placesStart
    if string match -q "#placesStart*" -- $line
        set capture true
        continue
    end

    # Check if the line starts with #placesEnd
    if string match -q "#placesEnd*" -- $line
        set capture false
        break
    end

    # If the capture flag is true, add the line to the list
    if $capture
        set -a captured_lines $line
    end
end

# Check if there are any captured lines
if not set -q captured_lines[1]
    echo "No lines found between #placesStart and #placesEnd."
    exit 1

set selected_line (printf '%s\n' $captured_lines | fzf --height 40% --reverse --prompt="Select a line: ")

# Check if the user selected something
if test -n "$selected_line"
    # If the selected line starts with "#", remove the "#"
    if string match -q "#*" -- $selected_line
        set selected_line (string replace -r '^#' '' -- $selected_line)
    end

    # Add the selected line (without "#") to each line in captured_lines that doesn't start with "#"
    set updated_lines
    for line in $captured_lines
        if not string match -q "#*" -- $line
            set -a updated_lines "$selected_line$line"
        else
            set -a updated_lines $line
        end
    end

    # Print the updated lines
    echo "Updated lines:"
    for line in $updated_lines
        echo $line
    end
else
    echo "No selection made."
end

Explanation of Changes:

  1. if string match -q "#*" -- $selected_line: Checks if the selected line starts with #.

  2. set selected_line (string replace -r '^#' '' -- $selected_line): Removes the # character from the beginning of the selected line using string replace.

  3. set updated_lines: Initializes an empty list to store the updated lines.

  4. for line in $captured_lines: Iterates over each line in the captured_lines list.

    • If the line doesn't start with #, it appends the modified selected_line to the beginning of the line.

    • If the line starts with #, it adds the line as-is to the updated_lines list.

  5. echo "Updated lines:": Prints the updated lines after processing.
